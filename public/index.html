Introduction

This documentation is generated from normal Markdown files using docbox, please check it for more details.

To access this API, a token should be provided. It can be use as req.body.token and also query string endpoint/?token=your_token or even with headers request.

Access token example

https://domain.com/{endpoint}?token=your-access-token
const Authorization = your-access-token;
axios.get('endpoint', { headers: { Authorization }})
  .then((res) => {
    console.log(res);
  })
  .catch((err) => {
    console.log(err);
  });
axios.post('endpoint', {
    token: your-token-access
  })
  .then((res) => {
    console.log(res);
  })
  .catch((err) => {
    console.log(err);
  });

Sign Up

A user must be have a token to use the API, and to get one they should registered on database. Use the signup endpoint to registered and signin to get access token. A token will expired in 3 days.

Property Description
username Use a uniques username
password Any string is acceptable, and it'll encrypt with bcrypt package
POST
/api/v1/signup

Example request

axios.post('endpoint', {
    username: "Doe",
    password: "1234"
  })
  .then((res) => {
    console.log(res);
  })
  .catch((err) => {
    console.log(err);
  });

Example response

{
  "success": true,
  "message": "Successful create a new user."
}

HTTP Response

HTTP 201

Failed to Sign Up example response

{
  "success": false,
  "message": "Username is already taken"
}

Sign In

After Sign in, you'll get the token provided by token key object on ajax response. Remember, this token will be expired in 3 days since you loggedin.

Property Description
username Your unique username
password Your password (it'll decrypted on server)
POST
/api/v1/signin

Example request

axios.post('endpoint', {
    username: "Doe",
    password: "1234"
  })
  .then((res) => {
